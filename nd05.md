## ポリシーの作成例

### ポート開放の基本方針

* ssh (tcp 22) 宛のINPUTをACCEPT


```
cd ~
nano fw1.sh
```


```bash
#!/bin/bash
#!/bin/bash

# 設定の初期化
sudo iptables -F
sudo iptables -X
sudo iptables -t nat -F
sudo iptables -t nat -X

# ssh サーバへのINPUTを許可する
sudo iptables -A INPUT -i eth0 -p tcp --dport 22 -j ACCEPT

# それ以外の入力は拒否
sudo iptables -A INPUT  -i eth0 -p tcp -j DROP

# 設定の確認
sudo iptables -L
sudo iptables -t nat -L

```
