## ポリシーの作成例

### ポート開放の基本方針

* http をACCEPT
* 同じ班からの ssh をACCEPT
* 他の班からのsshはDROP

### デフォルト否定

FWルータのルールの最後に以下を設定

* INPUT	チェーンはすべて DROP
* FOWARD チェーンはすべてDROP
* OUTPUT チェーンはすべてACCETP
