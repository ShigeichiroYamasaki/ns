### ポインタと間接演算子

```bash
nano pointer.c
```


```c
#include <stdio.h>
int main()
{
  int a = 100;
  int *p;
  p = &a;  /* ポインタpに変数aのアドレスを代入する*/

  return 0;
}
```

```bash

```


### ポインタのポインタ

```c
#include <stdio.h>
int main(void){
    int a = 0;
    int *p;   /*整数へのポインタ*/
    int **pp; /*整数へのポインタのポインタ*/
    p=&a;
    pp=&p;
    printf("**pp=%d\n", **pp);
    return 0;
}

```
