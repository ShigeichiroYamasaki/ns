### 

#### tanimoto係数を計算するメソッド

```ruby
# tanimoto係数メソッド(数値の配列だけの比較をする)

def tanimoto(a,b)
  na=a.count(1)
  nb=b.count(1)
  azb=a.zip(b)
  nab=azb.count([1,1])
  nab/(na+nb-nab).to_f
end

```

#### 映画と映画のtanimoto係数を求める


```ruby
tanimoto(m[0],m[1])
tanimoto(m[0],m[2])
tanimoto(m[0],m[3])
tanimoto(m[1],m[2])
```

### 拡張したtanimoto係数

$ \frac{\sum {(x_{ak} * x_{bk})}}{\sum x_{ak}^2+ \sum x_{bk}^2 - \sum (x_{ak} * x_{bk})} $


![](http://chart.apis.google.com/chart?cht=tx&chl=T%28A%2CB%29%3D%5Cfrac%7B%5Csum%28x_%7BAk%7D%2Ax_%7BBk%7D%29%7D%7B%5Csum%20x%5E2_%7BAk%7D%20%2B%20%5Csum%20x%5E2_%7BBk%7D%20-%20%5Csum%20%28x_%7BAk%7D%2Ax_%7BBk%7D%29%7D)

```ruby
def tanimoto2(a,b)
  sa=a.drop(1).map{|x|x**2}.sum
  sb=b.drop(1).map{|x|x**2}.sum
  sab=a.zip(b).drop(1).map{|x|x[0]*x[1]}.sum
  sab/(sa+sb-sab).to_f
end
```
